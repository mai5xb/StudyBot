[FAQ]
What is a variable?
A variable stores data in memory. Example: x = 5

What is a loop?
A loop repeats code. Python uses 'for' and 'while' loops.

How do I define a function?
Use 'def' and a name. Example:
def greet():
    print("Hello!")

What is a conditional?
Conditionals use 'if', 'elif', and 'else' to make decisions.

What is a syntax error?
An error in code structure. Check colons, parentheses, and indentation.

What is a list?
A list stores multiple items. Example: fruits = ['apple', 'banana']

How do I read a file?
Use:
with open('file.txt', 'r') as f:
    content = f.read()

What is debugging?
Finding and fixing errors in your code.

How do I prepare for exams?
Review concepts, practice problems, and explain topics to others.

What is recursion?
A function calling itself to solve a smaller problem.
